## 🌟 About the Application
Batch Agent (formerly CS_Compiled) is a native C# Windows application that provides a sleek, dark-themed interface for processing large volumes of documents. Whether you need to convert hundreds of Word files to PDF or manage complex directory structures, Batch Agent handles the "heavy lifting" so you don't have to.

## 💡 Our Philosophy
In today's digital landscape, many basic Windows operations—like batch converting Word documents to PDF—are locked behind paywalls or subscription models in the Microsoft Store and online tools. 

**We believe these native operations should be genuinely free to all users.** 

Batch Agent is an open-source initiative to provide these essential daily tools without the price tag, ensuring every user has access to professional-level automation.

## ✨ Key Features
- **High-Speed Conversion**: 
  - **Word to PDF**: Efficiently convert DOC/DOCX files.
  - **PDF to Word**: Extract editable documents from PDF payloads.
  - **PowerPoint to PDF**: Transform presentations into portable documents.
- **Flexible Processing Modes**:
  - **Individual Mode**: Process files within a specific folder.
  - **Batch Mode**: Recursively process all subfolders while maintaining the directory structure.
- **Modern UI/UX**:
  - Premium dark-themed interface.
  - Sidebar navigation with smooth animations.
  - Real-time progress tracking with detailed status labels.
- **Secure Processing**: All operations happen locally on your machine. Your data never leaves your computer.

## 🚀 How to Use
1. **Launch**: Open `AdvancedBatchConverter.exe`.
2. **Select Mode**: Choose between "File Conversion" or "File Management" from the sidebar.
3. **Configure Input**:
   - Select the **Source File Format**.
   - Select the **Target File Format**.
   - Choose your processing mode (Individual or Batch).
   - Click **Browse Import** to select your source folder.
4. **Configure Output**:
   - Click **Browse Export** to select where you want the processed files to be saved.
5. **Execute**: Click the **EXECUTE BATCH CONVERSION** button and watch the progress bar!

## 💻 System Requirements
- **OS**: Windows 10 or 11
- **Software**: Microsoft Office (Word/PowerPoint) must be installed for certain document conversions.
- **Framework**: .NET Framework 4.7.2 or higher.
